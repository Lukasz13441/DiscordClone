# DiscordClone

Klon aplikacji podobnej do Discord — czat / komunikator/web-aplikacja (frontend + backend).

# Opis projektu

DiscordClone to prosty klon platformy czatowej (podobnej do Discorda).  
Umożliwia: 
- tworzenie serwerów i kanałów
- wysylanie i odbieranie wiadomości w czasie rzeczywistym
- reakcje emoji
- system użytkowników i profili. 

# Funkcje / możliwości

### Użytkownicy
- Rejestracja konta
- Logowanie / wylogowanie
- Profil użytkownika
- Avatar użytkownika
- BIO użytkownika

### Serwery i kanały
- Tworzenie serwerów
- Automatyczny kanał `general`
- Tworzenie kanałów tekstowych oraz głosowych
- Kanały prywatne (1-na-1 – friendship)
- Przełączanie między kanałami

### Wiadomości (SignalR)
- Wysyłanie wiadomości w czasie rzeczywistym
- Oddzielne grupy SignalR dla każdego kanału
- Historia wiadomości z bazy danych
- Edycja wiadomości
- Usuwanie wiadomości

### Reakcje
- Reakcje emoji do wiadomości
- Dodawanie / usuwanie reakcji
- Licznik reakcji
- Synchronizacja reakcji w czasie rzeczywistym

# Instalacja / Uruchomienie

1. Sklonuj repozytorium
2. Otwórz projekt w Visual Stuido
3. Uruchom aplikację: Ctrl+F5
